## 本地开发

### 环境准备

- 安装 [Node.js](https://nodejs.org/en)

### 操作步骤

- 安装依赖

```sh
yarn install
```

- 启动 Dev Server

```sh
yarn run dev
```

- 在浏览器访问 http://localhost:3000

### 打包方式

- 运行打包

```sh
yarn build:client
```
